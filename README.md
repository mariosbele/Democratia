# Democratia 🗳️

> Η φωνή σου στη δημοκρατία.

Διαδραστικό πρωτότυπο (frontend) της εφαρμογής **Democratia** — μιας πλατφόρμας όπου οι
κυβερνήσεις/δήμοι ανεβάζουν τα ψηφίσματα (νομοσχέδια) ακριβώς όπως στο κοινοβούλιο, και οι
πολίτες ψηφίζουν **ανώνυμα** (ΝΑΙ / ΟΧΙ / ΠΑΡΩΝ), σχολιάζουν επώνυμα στην κοινότητα και
παρακολουθούν τα αποτελέσματα.

Βασισμένο στα έγγραφα **Business Requirements v1.1** και **App Design**.

## ✨ Βασικά χαρακτηριστικά

- **Σύνοψη AI κάθε ψηφίσματος** — αντί για το πλήρες νομικό κείμενο, εργαλείο τεχνητής
  νοημοσύνης εξάγει τα βασικά σημεία που πρέπει να γνωρίζει ένας πολίτης (TL;DR, βασικά σημεία,
  «τι σημαίνει για εσάς»).
- **Είσοδος / Εγγραφή** μέσω Ταυτότητας + Taxisnet (προσομοίωση KYC), δημιουργία & επαναφορά
  διαπιστευτηρίων.
- **Αρχική σελίδα** με επιλογή κοινωνίας (Αθήνα/Θεσσαλονίκη), φίλτρα κατηγοριών, ανοιχτά
  ψηφίσματα & αποτελέσματα.
- **Ψηφοφορία** ΝΑΙ / ΟΧΙ / ΠΑΡΩΝ — ανώνυμη και **αμετάκλητη**, με γραφήματα αποτελεσμάτων και
  χρονοδιάγραμμα ανακοινώσεων (έως 3 φάσεις).
- **Κοινότητα σχολίων** — επώνυμα σχόλια (1 ανά ψήφισμα, έως 1000 χαρακτήρες), «Συμφωνώ» (likes),
  παρακολούθηση (follow) πολιτών & πολιτικών.
- **Λογαριασμός** — Ακολουθείτε, Ιστορικό, Ρυθμίσεις, Q&A, Απόρρητο (GDPR), Ειδοποιήσεις.
- Σχεδιασμός **mobile-first** με «πλαίσιο κινητού» στο desktop. UI στα Ελληνικά.

## 🛠️ Τεχνολογίες

React 18 · Vite · React Router · Tailwind CSS

## 🚀 Εκτέλεση

```bash
npm install
npm run dev      # http://localhost:5173
```

Build για παραγωγή:

```bash
npm run build
npm run preview
```

## 📁 Δομή

```
src/
  data/mockData.js        # Ψηφίσματα (με AI συνόψεις), χρήστες, σχόλια, ειδοποιήσεις
  context/AppContext.jsx  # Κατάσταση (ψήφοι, σχόλια, likes, follows) + localStorage
  components/             # Επαναχρησιμοποιήσιμα UI (κάρτες, header, nav, εικονίδια)
  screens/                # Οθόνες της εφαρμογής
  lib/utils.js            # Βοηθητικές συναρτήσεις
  lib/api.js              # (προαιρετικό) client για το backend API
server/                   # Backend (Node.js + SQLite) — δες server/README.md
```

## 🛠️ Backend

Στον φάκελο [`server/`](./server) υπάρχει backend (Node.js + SQLite, **χωρίς εξωτερικά
dependencies**) που:

- **Διασυνδέει αυτόματα** τα ψηφίσματα με τη **Βουλή** (ρυθμιζόμενο feed) και το
  **Ευρωκοινοβούλιο** (European Parliament Open Data API v2).
- Αντλεί και αποθηκεύει τα **επίσημα αποτελέσματα** (ΥΠΕΡ/ΚΑΤΑ/ΑΠΟΧΗ) κάθε ψηφοφορίας.
- Καταγράφει **ανώνυμες & αμετάκλητες** ψήφους πολιτών και ανακοινώνει αποτελέσματα σε φάσεις.
- Εκθέτει REST API συμβατό με το σχήμα του frontend (`/api/bootstrap`).

```bash
cd server && npm start    # http://localhost:8080
```

Για ανέβασμα online (Render / Railway / Docker) με ενεργό live συγχρονισμό, δες
[`server/DEPLOY.md`](./server/DEPLOY.md).

## ⚠️ Σημειώσεις πρωτοτύπου

- Τα δεδομένα του frontend είναι **mock** (localStorage)· το backend στο `server/`
  παρέχει πραγματική βάση (SQLite) και αυτόματο συγχρονισμό με Βουλή/Ευρωκοινοβούλιο.
- Η ταυτοποίηση μέσω **Taxisnet** είναι προσομοίωση — η πραγματική υλοποίηση απαιτεί συνεργασία
  με την επίσημη κρατική βάση δεδομένων.
- Ο live συγχρονισμός απαιτεί εξερχόμενη δικτυακή πρόσβαση στις πηγές ανοιχτών δεδομένων.
