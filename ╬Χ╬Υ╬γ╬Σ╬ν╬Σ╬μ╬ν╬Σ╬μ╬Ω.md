# 📦 Οδηγίες Εγκατάστασης — Democratia (για εσένα)

Απλά βήματα, χωρίς τεχνική ορολογία. Διάλεξε **μία** από τις δύο επιλογές.

---

## ✅ Επιλογή Α — Μόνο η ιστοσελίδα (δωρεάν, πιο απλό)

Δείχνει τα πάντα (ψηφίσματα, συνόψεις AI, πλήρες κείμενο, αποτελέσματα,
σύνδεση με δοκιμαστικούς χρήστες). Ιδανικό για επίδειξη. **Κόστος: 0€.**

**Τι χρειάζεσαι:** έναν λογαριασμό GitHub και έναν Vercel (δωρεάν).

1. **Ανέβασε τον κώδικα στο GitHub.** Αποσυμπίεσε το zip στη ρίζα του project και
   ανέβασέ το (commit/push), όπως κάνεις ήδη.
2. **Σύνδεσε το Vercel με το GitHub.** Στο [vercel.com](https://vercel.com):
   *Add New → Project → διάλεξε το repository → Deploy*. Σε ~1 λεπτό είναι online.
3. **Τέλος.** Θα πάρεις μια διεύθυνση τύπου `https://democratia.vercel.app`.

**Πώς μπαίνεις:** στην οθόνη σύνδεσης βάλε όνομα χρήστη `mbelechris` και κωδικό
`Demo!2024` και μπαίνεις απευθείας (η ταυτοποίηση γίνεται μέσω Taxisnet — δεν
χρειάζεται κωδικός OTP).

> Σε αυτή την επιλογή τα δεδομένα είναι αυτά που ετοιμάσαμε και η ψήφος κάθε
> επισκέπτη μένει στη δική του συσκευή. Για ζωντανή ανανέωση & κοινό μέτρημα,
> δες την Επιλογή Β.

## ✅ Επιλογή Β — Ιστοσελίδα + Server (πλήρης λειτουργία)

Προσθέτει: αυτόματα δεδομένα Ευρωκοινοβουλίου, κοινό μέτρημα ψήφων, πραγματική
σύνδεση/λογαριασμούς, σελίδα διαχείρισης για τη Βουλή. **Κόστος: 0€** (δωρεάν
πακέτο Render) ή ~5€/μήνα για κάτι πιο σταθερό.

1. Κάνε πρώτα τα βήματα της **Επιλογής Α** (ιστοσελίδα στο Vercel).
2. **Σήκωσε τον server:** στο [render.com](https://render.com) → *New → Blueprint*
   → διάλεξε το repo → *Apply*. Διαβάζει μόνο του τις ρυθμίσεις (`render.yaml`).
3. Θα πάρεις διεύθυνση τύπου `https://democratia-api.onrender.com`. Άνοιξέ την με
   `/api/health` στο τέλος — αν δεις `{"status":"ok"}`, ο server παίζει.
4. **Σύνδεσε τα δύο:** στο Vercel → *Settings → Environment Variables* → πρόσθεσε
   `VITE_API_URL` με τιμή τη διεύθυνση του server (χωρίς `/` στο τέλος) → *Redeploy*.
5. **Διαχείριση Βουλής:** άνοιξε `https://<ο-server-σου>/admin`. Τον κωδικό
   (`ADMIN_TOKEN`) τον δημιουργεί αυτόματα το Render — θα τον βρεις στο
   Render → το service → *Environment*.

Αναλυτικός οδηγός: **`server/ΟΔΗΓΟΣ.md`**.

## 💻 (Προαιρετικό) Δοκιμή στον υπολογιστή σου

Χρειάζεται **Node.js έκδοση 22+** (δωρεάν, από [nodejs.org](https://nodejs.org)).

**Η ιστοσελίδα:**
```bash
npm install
npm run dev        # άνοιξε http://localhost:5173
```

**Ο server (σε άλλο παράθυρο):**
```bash
cd server
npm start          # http://localhost:8080
```

Για να μιλήσει η τοπική ιστοσελίδα με τον τοπικό server, φτιάξε ένα αρχείο
`.env.local` στη ρίζα με: `VITE_API_URL=http://localhost:8080`
